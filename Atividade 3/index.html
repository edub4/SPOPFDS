Guia de Comandos Básicos do Git

Resumo prático dos principais comandos utilizados no Git.

1. Introdução aos commits

git init
git add .
git commit -m "Primeiro commit"
  • git init: inicia o repositório
  • git add .: adiciona arquivos
  • git commit: salva uma versão

2. Referências relativas

git checkout HEAD^
git checkout HEAD~2
  • HEAD: commit atual
  • ^: volta 1 commit
  • ~2: volta 2 commits

Branches (Movendo trabalho)

git branch nova-branch
git checkout nova-branch
git merge main
  • Criar nova branch
  • Trocar de branch
  • Juntar alterações

Cherry-pick

git cherry-pick HASH

Copia um commit específico para outra branch.

Comandos úteis

git log
git status
git diff
  • Ver histórico
  • Ver status
  • Ver diferenças

Rebase

git rebase main
git rebase -i HEAD~3

Organiza e reescreve o histórico.

Temas avançados

git reset --hard HEAD~1
git revert HEAD
  • reset: apaga commits
  • revert: desfaz sem apagar

Rebase avançado

git rebase -i HEAD~5

Edita vários commits.

Push & Pull

git remote add origin URL
git push -u origin main
git pull origin main

Histórico divergente

git pull --rebase
git merge branch

Remoto avançado

git fetch
git push origin branch

Push Main

git push origin main
